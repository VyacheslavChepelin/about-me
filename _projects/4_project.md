Одностраничное веб-приложение для управления структурой покерного турнира. Помогает следить за раундами, блайндами, анте и временем до перерыва — всё в полноэкранном режиме.

### Технологии

- **HTML5, CSS3, JavaScript (ES6)** — полностью самописный код без фреймворков и библиотек.  
  Адаптивная вёрстка с `clamp()`, Flexbox, плавные переходы.
- **GitHub Actions** — автоматическая сборка единого HTML-файла при каждом пуше в `main`.
- **Node.js** — скрипт `build.js` вставляет содержимое `styles.css` и `script.js` в шаблон `template.html`, формируя готовый `poker_timer.html`.
- **JSON** — структура раундов (длительность, блайнды, анте) и список перерывов хранятся в формате JSON, их можно редактировать прямо в интерфейсе.

### Основные возможности

- Настраиваемые раунды: длительность, блайнды, анте.
- Автоматические перерывы после заданных раундов.
- Управление с клавиатуры:  
  <kbd>Пробел</kbd> — пауза / продолжение,  
  <kbd>↑</kbd><kbd>↓</kbd> — изменение числа активных игроков,  
  <kbd>←</kbd><kbd>→</kbd> — изменение количества стеков.
- Загрузка фонового изображения.
- Расчёт среднего стека на основе общего количества фишек и числа игроков.
- Финишный экран с надписью «Турнир завершён».
